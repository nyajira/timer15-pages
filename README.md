# 時感密度

時感密度は、今日を少し長く、濃く感じるための時間リマインダーアプリです。

子どもの頃、1日はもっと長く感じられました。  
それは、時間の中に小さな区切りや気づきがたくさんあったからかもしれません。

時感密度では、3分から30分のあいだでランダムなタイミングにメッセージが届きます。  
予測できない小さな合図が、流れるように過ぎていく時間から、今この瞬間へ意識を戻すきっかけになります。

仕事、勉強、読書、家事、休憩。  
何気ない日常の中に小さな節目をつくることで、1日の手触りを取り戻すことを目指します。

忙しく過ぎていく毎日に、少しだけ時間の輪郭を。

---

## Features

- ランダム通知（3〜30分）
- シンプルで集中を妨げない設計
- 6言語対応
  - 日本語
  - English
  - 中文
  - Indonesia
  - Español
  - Português
